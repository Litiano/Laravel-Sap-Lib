# Lib para conectar com SAP via DI-API

## Instruções
### Adicionar as seguintes configurações em seu arquivo .env
* SAP_DB_SERVER_TYPE
. dst_MSSQL = 1
. dst_DB_2 = 2
. dst_SYBASE = 3
. dst_MSSQL2005 = 4
. dst_MAXDB = 5
. dst_MSSQL2008 = 6
. dst_MSSQL2012 = 7
* SAP_SERVER_ADDRESS
* SAP_DB_NAME
* SAP_LICENSE_SERVER
* SAP_USER
* SAP_PASSWORD
* SAP_SQL_PORT
* SAP_DB_USER
* SAP_DB_PASSWORD
